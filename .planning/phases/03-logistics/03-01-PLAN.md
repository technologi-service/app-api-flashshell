Implement three exported functions:

**`getPickupList()`** -- uses `db` (Drizzle HTTP client, no transaction needed):
- Query orders with `status IN ('preparing', 'ready_for_pickup') AND courier_id IS NULL`
- For each order, join with `order_items` and `menu_items` to get item name + quantity
- Return array of `{ id, status, items: [{ name, quantity }], totalAmount, deliveryAddress, createdAt }`
- Use raw SQL via `db.execute(sql\`...\`)` for the join query:
```sql
SELECT o.id, o.status, o.total_amount, o.delivery_address, o.created_at,
       oi.quantity, mi.name AS item_name
FROM orders o
JOIN order_items oi ON oi.order_id = o.id
JOIN menu_items mi ON mi.id = oi.menu_item_id
WHERE o.status IN ('preparing', 'ready_for_pickup') AND o.courier_id IS NULL
ORDER BY o.created_at ASC
```
Aggregate rows in JS: group by order id, collect items array per order.

**`getOrderDetail(orderId: string, courierId: string)`** -- uses `db`:
- Query single order by id with items join (same join pattern as above but `WHERE o.id = $1`)
- Access control: if order has `courier_id` set and it does not match `courierId`, AND status is NOT `preparing`/`ready_for_pickup`, return `{ found: false, reason: 'FORBIDDEN' }`
- If no order found, return `{ found: false, reason: 'NOT_FOUND' }`
- Return `{ found: true, order: { id, status, items, totalAmount, deliveryAddress, courierId, createdAt } }`

**`advanceOrderStatus(orderId: string, courierId: string, newStatus: 'picked_up' | 'delivered')`** -- uses `txPool` (transaction with SELECT FOR UPDATE):

For `picked_up`:
1. `BEGIN`
2. `SELECT id, status, courier_id FROM orders WHERE id = $1 FOR UPDATE` -- locks row
3. Validate `status === 'ready_for_pickup'` -- else ROLLBACK, return `{ ok: false, error: 'INVALID_TRANSITION' }`
4. Validate `courier_id IS NULL` -- else ROLLBACK, return `{ ok: false, error: 'ALREADY_CLAIMED' }`
5. Check one-active-order: `SELECT id FROM orders WHERE courier_id = $1 AND status = 'picked_up'` with `[courierId]` -- if rows > 0, ROLLBACK, return `{ ok: false, error: 'COURIER_BUSY' }`
6. `UPDATE orders SET status = 'picked_up', courier_id = $1, updated_at = NOW() WHERE id = $2` with `[courierId, orderId]`
7. `SELECT pg_notify('flashshell_events', $1::text)` with `JSON.stringify({ channel: \`order:${orderId}\`, event: 'order_picked_up', orderId, courierId })`
8. `SELECT pg_notify('flashshell_events', $1::text)` with `JSON.stringify({ channel: 'control', event: 'order_picked_up', orderId, courierId })`
9. `COMMIT`
10. Return `{ ok: true }`

For `delivered`:
1. `BEGIN`
2. `SELECT id, status, courier_id FROM orders WHERE id = $1 FOR UPDATE`
3. Validate `status === 'picked_up'` AND `courier_id === courierId` -- else ROLLBACK, return `{ ok: false, error: 'FORBIDDEN' }`
4. `UPDATE orders SET status = 'delivered', updated_at = NOW() WHERE id = $1`
5. `SELECT pg_notify('flashshell_events', $1::text)` with `JSON.stringify({ channel: \`order:${orderId}\`, event: 'order_delivered', orderId })`
6. `SELECT pg_notify('flashshell_events', $1::text)` with `JSON.stringify({ channel: 'control', event: 'order_delivered', orderId })`
7. `COMMIT`
8. Return `{ ok: true }`

Wrap all in try/catch with ROLLBACK in catch, `client.release()` in finally (same pattern as consumer/service.ts).

**3. Create `src/plugins/logistics/index.ts`:**

```typescript
import { Elysia } from 'elysia'
import { authPlugin } from '../auth/index'
import { requireRole } from '../auth/require-role'
import { AdvanceStatusBody } from './model'
import { getPickupList, getOrderDetail, advanceOrderStatus } from './service'

export const logisticsPlugin = new Elysia({ name: 'logistics', prefix: '/logistics' })
  .use(authPlugin)
  .use(requireRole('delivery'))
  .get('/orders/ready', () => getPickupList(), { auth: true })
  .get(
    '/orders/:id',
    async ({ params, user, status }) => {
      const result = await getOrderDetail(params.id, user.id)
      if (!result.found) {
        const code = result.reason === 'NOT_FOUND' ? 404 : 403
        return status(code, {
          error: result.reason,
          message: result.reason === 'NOT_FOUND'
            ? `Order ${params.id} not found`
            : 'Not authorized to view this order'
        })
      }
      return result.order
    },
    { auth: true }
  )
  .patch(
    '/orders/:id/status',
    async ({ params, body, user, status }) => {
      const result = await advanceOrderStatus(params.id, user.id, body.status)
      if (!result.ok) {
        const statusCode = result.error === 'NOT_FOUND' ? 404
          : result.error === 'FORBIDDEN' ? 403
          : 409  // ALREADY_CLAIMED, COURIER_BUSY, INVALID_TRANSITION
        return status(statusCode, {
          error: result.error,
          message: result.error === 'ALREADY_CLAIMED' ? 'Order already claimed by another courier'
            : result.error === 'COURIER_BUSY' ? 'You already have an active delivery'
            : result.error === 'INVALID_TRANSITION' ? 'Cannot transition from current status'
            : result.error === 'NOT_FOUND' ? 'Order not found'
            : 'Not authorized'
        })
      }
      return { success: true, status: body.status }
    },
    { auth: true, body: AdvanceStatusBody }
  )
```
